# Noise

An experiment with Twitter, Processing.js and Socket.io.

## Installation

After checking out:

	$ cd noise/
	$ npm install .

That should do it. All dependencies will be installed by NPM.

## Starting Up the app

It's an ExpressJS app, so all the usual applies.

	$ node app.js
